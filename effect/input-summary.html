input的提示语，只需要加入 placeholder=""

鼠标点击进去后有黄色的边框，只需要css中加入outline:none;

input的提示语的颜色修改

::-webkit-input-placeholder { /* WebKit browsers */
color: #ec8054;
}
:-moz-placeholder { /* Mozilla Firefox 4 to 18 */
color: #ec8054;
}
::-moz-placeholder { /* Mozilla Firefox 19+ */
color: #ec8054;
}
:-ms-input-placeholder { /* Internet Explorer 10+ */
color: #ec8054;
}

禁止输入，只需要加上readonly="readonly"

隐藏，只需要加上type="hidden"

input不能换行，我们可以用模仿的div来实现换行，只需要加入contenteditable="true"就可以实现输入

模仿的div也想加上提示语
